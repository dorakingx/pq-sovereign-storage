Post-Quantum Sovereign Privacy Storage

A hackathon prototype for a future-proof private data layer on 0G — combining post-quantum-friendly commitments, encrypted storage payloads, and on-chain verifier registry.

Rust 0G Storage 0G Chain Privacy & Sovereign Infrastructure

What it does

  • Encrypts local files before upload to 0G Storage
  • Generates SHAKE256-based state commitments over encrypted payloads
  • Uploads, downloads, and decrypts with user-held encryption keys
  • Records verifiable commitments on chain via PqspVerifier.sol

Quick start

cargo build
cargo run -- upload ./example.txt
cargo run -- verify ./example.txt.pqsp-receipt.json
cargo run -- download <MERKLE_ROOT>
cargo run -- decrypt <DOWNLOADED_FILE> <KEY_HEX>
